## 搬瓦工是什么？BWG 的定位一句话说清楚

搬瓦工（BandwagonHost）是加拿大 IT7 Networks 旗下的 VPS 品牌，KVM 虚拟化架构，自研 KiwiVM 控制面板，面向自管理用户——也就是说，不提供托管服务，所有配置都自己来，但价格因此压得很低。

它对国内用户的核心吸引力在于线路。搬瓦工在洛杉矶 DC6、DC9、香港、日本大阪等机房接入了 CN2 GIA（中国电信顶级精品线路），这条线路在晚高峰表现远比普通国际线路稳，丢包率低，延迟也更可控。做了十几年，口碑在中文 VPS 圈子里属于比较扎实的那种。

## 三种线路的区别，这一段最重要

很多人买 BWG 买错套餐，根源在于没搞清楚 CN2 GT 和 CN2 GIA 的区别。

**CN2 GT（Global Transit）**：CN2 系列里价格较低的那档，走中国电信的骨干网，但跟普通国际线路共享带宽。晚高峰时经常有拥堵，延迟抖动明显。KVM 套餐走的就是这类线路（部分机房），适合对速度要求不高、就是要个便宜服务器跑脚本或学 Linux 的。

**CN2 GIA（Global Internet Access）**：顶级精品线路，独享带宽，三网直连——电信、联通、移动全程 GIA 回程。这条线路的成本极高，市场上能稳定提供 CN2 GIA VPS 的商家本来就不多。搬瓦工在洛杉矶 DC6 和 DC9 部署了多条 10G 级别的 CN2 GIA 链路，晚高峰丢包率控制得很好。

**香港 CN2 GIA**：物理上离大陆最近，延迟可以压到 5-50ms 区间，华南地区用户几乎感觉不出和本地服务器的差距。但价格是这三档里最贵的，个人用户大多不是刚需。

一句话选择参考：学习用 → KVM；建站/代理/AI 部署 → CN2 GIA-E；实时业务/游戏加速/企业应用 → 香港 CN2 GIA。

## 常见问题 FAQ

**Q：BWG 搬瓦工适合新手吗？**

适合，但要接受它是自管理模式——没有客服帮你装软件，遇到问题靠文档和自己搜索。KiwiVM 控制面板操作比较直观，重装系统、迁机房都是几次点击的事，入门难度不高。

**Q：CN2 GIA-E 和 CN2 GIA 有什么区别？**

命名上 GIA-E 是 CN2 GIA Ecommerce 的缩写，实际上两者线路质量差不多，都是 CN2 GIA 顶级精品线路，主要区别在套餐配置和可选机房数量。CN2 GIA-E 套餐可选机房更多，性价比更高，是大多数人的首选。

**Q：搬瓦工会自动续费吗？**

不会。搬瓦工不保存信用卡信息，不做自动扣款，每次续费需要手动操作。到期前会发邮件提醒。

**Q：IP 被封了怎么办？**

KiwiVM 后台可以申请更换 IP，一般情况下这个功能可用。部分套餐支持每两周免费换一次 IP（比如电商 SLA 套餐）；常规套餐换 IP 可能需要额外费用，具体以官网当期政策为准。

**Q：支付宝可以直接付款吗？**

可以，直接在结算页选支付宝，跳转扫码付款就行，流程很顺。
